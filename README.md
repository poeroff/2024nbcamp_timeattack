    프로젝트 개요 
    모든 웹사이트의 기본적인 인증을 구현하는 프로젝트로써 인증되지않은 사용자는 허락된 유저만 사용할 수 있는 사이트를 만듬과 동시에 예상하지 못한 예외처리를 할수 있는 기본적인 로직을 만드는 프로젝트인데.

    구현하며 중요하게 생각햇던 점 
    인증되지 않은 사용자는 접속이 안되게 하는것(구현못함)
    예상하지 못한 예외처리
    Jwt토큰을 사용하여 인증을 처리하는 미들웨어 작성
